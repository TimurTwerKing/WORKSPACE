Preguntas Frecuentes

¿Cuál es el costo del alquiler?

El costo del alquiler varía según el modelo de dron que desee alquilar. Por favor, consulte nuestra página web para conocer los precios actuales.

¿Es necesario tener experiencia previa en el manejo de drones?

No es necesario tener experiencia previa en el manejo de drones. Ofrecemos un breve tutorial sobre cómo manejar el dron antes de que lo alquile.

¿El alquiler incluye seguro?

Sí, el alquiler incluye un seguro básico que cubre daños accidentales. Si desea un seguro adicional, puede adquirirlo por un costo adicional.

¿Con cuánto tiempo de anticipación debo reservar el alquiler?

Le recomendamos que reserve su alquiler con al menos una semana de anticipación para garantizar la disponibilidad del dron que desea.

¿Puedo alquilar un dron para uso temporal?

Sí, puede alquilar un dron para uso temporal. Ofrecemos alquileres por día, semana o mes.

¿Se ofrecen descuentos o promociones especiales para alquileres a largo plazo?

Sí, ofrecemos descuentos y promociones especiales para alquileres a largo plazo. Por favor, consulte nuestra página web para conocer las ofertas actuales.
